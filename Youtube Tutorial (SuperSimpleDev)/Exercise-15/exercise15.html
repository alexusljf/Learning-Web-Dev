Position Absolute and Relative

Notes
Position Absolute also takes Left Top Right Bottom values
Position fixed elements are placed on the browser window, Absolute are placed on the page
Generally, elements written below will appear in front of elements written above
Using z-index: can set which element appear in front, higher val = more front, default = 0
Set a very high z-index value to make it the most front, like the top bar has z-index 100
Placing a Position Absolute inside a Position Fixed will place it relative on the fixed element, not the page
Position Relative still displays as normal, but if put an Absolute inside it will be Absolute on the Relative instead of the page
