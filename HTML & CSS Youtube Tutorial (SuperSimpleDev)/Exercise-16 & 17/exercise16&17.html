Position Absolute and Relative

Notes (16)
.className are Class Selectors, elementName{} are Element Selectors
Can target multiple class together using ,
.searchBar .toolTip to target the .toolTip class inside, don't need unique names for every tool-tip element
.sidebar-link img{} is targeting all img in .sidebar-link
justify-content:center and align-items: center can align horizontally and vertically
Use Position to create tool-tips (position at bottom right), it's just there but hidden when not hovered over
Opacity 0 -> 1 to hide and show tool-tips, can also use transitions for this
pointer-events: none to disable hovers on tool-tips;
whitespace: nowrap to prevent text from wrapping around

Notes (17)
Responsive design makes sure the website looks good on different window sizes
@media (max-width: 600px){} is a media query, the CSS here will only activate when max-width > 600px
CSS Specificity: if multiple styles are put on an element, the more specific element will take precedence
